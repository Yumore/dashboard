* 应用的很多界面没有很细致的搭建，界面比较粗糙，需要优化
* 小说的书单详情页，本来已经做了，但发现获取不到数据，得继续优化
* 对应网络数据有做缓存，但缓存没有区分GET和POST，单纯的根据网址缓存，还有缓存时间没有做设置
* 点击侧滑刷新小说，漫画和动画的功能，这个刷新做的很粗糙，完全是强刷，其实很有问题。
* 小说的阅读界面对于章节的跳转没有处理，进去都是从第一章开始阅读。
